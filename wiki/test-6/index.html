      2. Both forEach and Map take three element: value of the element, index of the element and the array object. Both return a callback function.       Filter creates a new array by removing items that do not belong to the array. Reduce takes all the element of an array and reduces them to a single value (for instance when working with mathematical operations). The callback of reduce takes an accumulator and current value as parameter and returns a single value.
